# Heart Disease Classification Project

This project uses machine learning to predict heart disease based on patient medical data.

## Dataset

The data comes from the Cleveland Heart Disease dataset from UCI Machine Learning Repository. It includes features like age, sex, chest pain type, blood pressure, cholesterol, and more.

## Files

- `end-to-end-heart-disease-classification.ipynb`: Main notebook with data exploration, preprocessing, model training, and evaluation.
- `data/heart-disease.csv`: The dataset used for training and testing.

## Models Used

- Logistic Regression
- K-Nearest Neighbors
- Random Forest

## How to Run

1. Install required libraries: pandas, numpy, matplotlib, seaborn, scikit-learn.
2. Open the notebook in Jupyter.
3. Run the cells in order.

## Evaluation

The goal is to achieve 95% accuracy in predicting heart disease.
